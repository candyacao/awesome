Awesome 是由 Docker+SpringBoot+Mybatis+thymeleaf 等技术实现的 Java 博客系统，博客模板是[@biezhi](https://github.com/biezhi)大神的开源项目[tale](https://github.com/otale/tale)，本来是一个 docker 和 springboot 的实战练习项目,如果觉得这个项目不错，请为它[点赞](https://github.com/candyacao/awesomenotes/stargazers)支持。
